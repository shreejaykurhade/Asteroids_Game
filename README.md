# Asteroids Game

A modern retro-styled asteroids game built with React and Vite, featuring local leaderboard persistence and multiple ship types.

## Features

- Classic asteroids gameplay with modern controls
- **Local Leaderboard**: Top 20 scores stored in browser localStorage
- **Ship Selection**: Choose from multiple ship types with unique stats
- **Responsive Design**: Works on desktop and mobile
- **Retro Aesthetic**: Neon colors and pixel-perfect styling

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment

Deploy to Vercel with one click or manually:

```bash
npm run build
vercel --prod
```

## Controls

- **Arrow Keys**: Rotate and thrust
- **Space**: Fire
- **Z**: Activate ship ability
